## How to compile and run
make sure you are in the same directory as the Makefile and project files. 
### using make:
```
$ make
$ ./smallsh
```

### or compile manually:
```
$ gcc -std=gnu99 -g -Wall -o smallsh smallsh.c
$ ./smallsh
```
